/**REVOLUTIONS OBJECT PRIMARY ID REQUIREMENTS**/
Format: int only. (ex.0010010010001)

/**PRIMARY ID DECODED**/
(Notes: No dashes. Dashes are just to temporarily explain the ID format.)
ex. 001-001-001-0001
Decoded: RevVersion-ObjectType-ObjectSubType-PersonalId

/**PRIMARY ID LEGEND: REVOLUTIONS VERSION**/
RevVersion:
	000 - October 2025

/**PRIMARY ID LEGEND: OBJECT TYPE**/
Table of Contents:
000 to 009 Classes
010 to 019 Skills
020 to 029 Runes
030 to 039 Items: Consumables
040 to 049 Items: Equipment
050 to 059 Items: Materials
060 to 99 (Unassigned. Reserved for System Objects)
(Notes: 100 and above is for User Objects. High error rate. These are subject to multiple edits, deletions, creation, deprecation.)
100 Story: Odyssey
102 Story: Sirius
